# React Skia

Universal React Skia

A shared Skia package that runs on ReactJS and React Native.

## Install

```sh
yarn
```

## Web App

```sh
cd packages/web
yarn start
```

## Mobile App

```sh
cd packages/mobile
yarn start
```
